# Peer to peer share file

### Make

```
cd p2p-project
make
```

### Clean build

```
make clean
```

Chạy server trước client sau

### Run Server

Vào bên trong thư mục TCP_Server
./server <Server_port>

### Run Client

Mỗi client có 1 folder riêng vào trong folder của mỗi client và sử dụng lệnh

```
./client <Server_IP> <Server_port> <Listen_port>
```
